# Password Strength Checker (Python)

A simple Python tool that rates password strength based on common security best practices.

## What it checks
- 12+ characters
- Uppercase letter
- Lowercase letter
- Number
- Special character
- Not a common password (basic list)

## How to run
```bash
python password_checker.py


Example

Input:
    •    password123

Output:
    •    Weak / Okay / Strong + tips to improve
